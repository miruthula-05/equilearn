# EquiLearn ♿📖

## Overview

EquiLearn is an accessible learning platform designed to support students with visual and hearing impairments. The system focuses on making educational content more inclusive by providing voice-based navigation and simplified learning resources.

The goal of EquiLearn is to reduce accessibility barriers in digital education and create a learning experience that supports diverse learners.

---

## Problem Statement

Many digital learning platforms are not designed with accessibility in mind. Visually impaired students often struggle to navigate educational platforms, while hearing-impaired learners may find it difficult to follow traditional audio-based learning resources.

This creates a significant barrier to inclusive education.

---

## Solution

EquiLearn provides an accessible interface that helps students interact with learning content using voice navigation and simplified educational materials.

The platform enables visually impaired users to navigate through voice commands and provides structured learning content that improves accessibility for all users.

---

## Features

* Voice-based navigation for visually impaired users
* AI-generated summarized learning content
* Accessible learning interface designed for inclusivity
* Structured educational materials for better understanding
